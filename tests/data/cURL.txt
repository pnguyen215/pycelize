3. TEMPLATE-BASED JSON GENERATION
===================================
$ curl -X POST \
  -F "file=@test_data/final_test.xlsx" \
  -F 'template={"person":{"name":"{name}","contact":"{email}"}}' \
  -F 'column_mapping={"name":"Name","email":"Email"}' \
  http://localhost:5050/api/v1/json/generate-with-template

Response:
{
  "data": {
    "download_url": "http://localhost:5050/api/v1/files/downloads/final_test_generated_template_20260130_102102.json",
    "file_size": 180,
    "total_records": 2
  },
  "message": "JSON file generated successfully with template",
  "status_code": 200
}

Generated File (final_test_generated_template_20260130_102102.json):
