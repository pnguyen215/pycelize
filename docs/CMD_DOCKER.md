# Build and run with Docker
```bash
make docker-build
```

```bash
make docker-run
```

# Or use Docker Compose
```bash
make docker-compose-up
```

# View logs
```bash
make docker-logs
```

# or
```bash
make docker-compose-logs
```

# Stop
```bash
make docker-stop
```
# or

```bash
make docker-compose-down
```

# Rebuild and restart
```bash
make docker-compose-rebuild
```

# Clean up
```bash
make docker-clean
```
